arcadia.internal.meval

meval

Syntax

  • (meval & stuff)

Description

Evaluates arguments in macro phase and injects result at macro site. Useful for manipulating code evaluation order without having to define named macros.

Arguments

  • stuff

defn-meval

Syntax

  • (defn-meval & preamble-and-meval-to-forms)

Description

Same as defn, except last form in preamble-and-meval-to-forms is evaluated during macro expansion, and the results spliced as code into the tail of the function definition. Useful for many things, such as automating the generation of specific arities for a function that would otherwise be inefficiently variadic.

Arguments

  • preamble-and-meval-to-forms
