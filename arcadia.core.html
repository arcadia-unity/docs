arcadia.core

Core Arcadia API Contains functionality to manipulate Unity scenes and attach clojure functions to Unity events. You will typically `(:use arcadia.core)` in your namespaces to access this functionality **PRE-ALPHA** this documentation and API are not finalized. Feel free to poke around but expect everything to change.

editor?

Syntax

  • (editor?)

Description

Returns true if called from within the editor. Notably, calls from the REPL are considered to be form within the editor

null-obj?

Syntax

  • (null-obj? x)

Description

Tests if an object is `nil` or not. Unity's notion of `nil` is complicated by the fact that they use custom null objects that evaluate to `true`.

Arguments

  • xObject

    the object to test

Examples

;; does the object have a camera?
(null-obj? (.GetComponent (object-named "Main Camera")
                          UnityEngine.Camera))

create-primitive

Syntax

  • (create-primitive prim)

Description

Creates a [GameObject](http://docs.unity3d.com/ScriptReference/GameObject.html "GameObject") with a primitive mesh renderer and appropriate collider. Equivalent to using the [GameObject](http://docs.unity3d.com/ScriptReference/GameObject.html "GameObject") → Create Other... menu items

Arguments

  • prim

    one of `:sphere` `:capsule` `:cylinder` `:cube` `:plane` or `:quad`

Examples

(create-primitive :cube)
;; => #<GameObject Cube>

;; create cube at random position
(let [cube (create-primitive :cube)]
  (set! (.position cube)
        (v3 (rand)
            (rand)
            (rand))))
;; create random primitive fired off into space!
(let [prim-kws [:sphere :capsule :cylinder :cube :plane :quad]
      prim (create-primitive (rand-nth prim-kws))
      rb (.AddComponent prim Rigidbody)]
  (.AddForce rb (Vector3. (rand 100)
                          (rand 100)
                          (rand 100))))

destroy

Syntax

  • (destroy target)
  • (destroy target t)

Description

Removes `target` from the scene. If called with `t`, removal happens after a timeout of `t` seconds.

Arguments

  • targetObject

    [Object](http://docs.unity3d.com/ScriptReference/Object.html "Object") to remove, a [GameObject](http://docs.unity3d.com/ScriptReference/GameObject.html "GameObject"), [Component](http://docs.unity3d.com/ScriptReference/Component.html "Component") or Asset

  • tDouble

    Number of seconds to wait before removing `target`

Examples

;; remove the main camera immediatley
(destroy (object-named "Main Camera"))
;; remove the main camera in five seconds
(destroy (object-named "Main Camera") 5)
;; remove all objects tagged "enemy"
(doseq [enemy (objects-tagged "dorun")]
  (destroy enemy))
;; testing hyperlinks 
(let [obj (object-named "set me")
      foos (objects-named "foo")]
  (if (< (count foos) 100)
    (destroy (object-named "Main Camera"))
    (doseq [f foos]
      (destroy f)
      (instantiate f (Vector3/Normalize (Vector3. 1 2 5))))))
;; map is lazy, so this will not work because it will not
;; evaluate unless forced to
(map destroy (objects-tagged "enemy"))

;; instead, use dorun or doall to force evaluation
(dorun (map destroy (objects-tagged "enemy")))
(doall (map destroy (objects-tagged "enemy")))

;; doseq is also useful   
(doseq [e (objects-tagged "enemy")]
  (destroy e))
